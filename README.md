# STM-32-BASED-MICROCONTROLLER-KIT
Designed a 2-layer PCB schematic for an STM32-based development board using KiCad.Developed and debugged embedded C firmware in STM32CubeIDE integrating GPIO, UART, ADC, and Timer peripherals.  Performed hardware bring-up, PCB assembly, and testing using oscilloscope and multimeter to validate circuit performance.
